# Upgrade 90 dias — Mechanistic Watchdog

Este repositorio consolida el trabajo original de Luis Cosio y añade un ciclo de mejoras equivalente a tres meses de continuidad técnica y editorial. El objetivo es mantener la línea conceptual del proyecto base y elevar la calidad de validación, presentación y trazabilidad.

Proyecto base: https://github.com/luiscosio/MechWatch.git.

## Continuidad con el proyecto original

- Se conserva el paquete Python para calibración, runtime y dashboard en `MechWatch/`.
- Se mantienen notebooks de estrés y exploración de datasets en `notebooks/`.
- Se respetan los insumos de evaluación (truthfulness y bio-defense) y sus umbrales.

## Mejoras incorporadas

- Sitio web en español con jerarquía editorial y lectura extendida.
- Gráficas basadas en datos reales, incluyendo boxplots generados desde JSON.
- Paquete Overleaf con figuras embebidas y bibliografía enlazada.
- Documentación unificada del flujo completo (web + investigación + pipeline).

## Entregables actuales

- Sitio Next.js listo para despliegue.
- Paper en `paper/main.tex` y zip listo para Overleaf.
- Scripts de generación de gráficas y pipeline.

## Alcance de continuidad

Este upgrade no reemplaza el proyecto original: lo amplía, lo unifica y lo deja listo para evolución continua sin romper compatibilidad con el repositorio base.
